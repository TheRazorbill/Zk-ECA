# issuer-proxy

Servidor Fastify que atua como ponte com o fluxo de emissao e verificacao de prova para o ecossistema Gov.br.

## Endpoints

- `GET /health`
- `POST /govbr/challenge`
- `POST /proof/issue`
- `POST /proof/verify`

## Scripts

- `pnpm dev`
- `pnpm build`
- `pnpm start`
- `pnpm lint`
- `pnpm check-types`
